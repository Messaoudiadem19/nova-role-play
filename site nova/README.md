# Nova Role Play Landing Page

هذا مشروع موقع تعريفي بسيط لسيرفر Nova Role Play الخاص بـ SAMP.

## المميزات
- تصميم حديث باللونين الأزرق والأسود
- عرض IP السيرفر
- زر نسخ IP
- زر Discord يفتح الرابط مباشرة
- قسم قواعد Role Play واضح ومنظم

## الهيكل
- index.html: الصفحة الرئيسية
- styles.css: تنسيق الموقع والألوان
- script.js: وظيفة نسخ IP

## طريقة العرض
يمكنك فتح الملف التالي مباشرة في المتصفح:

- [index.html](index.html)

أو تشغيل خادم محلي بسيط باستخدام Python:

```bash
python -m http.server 8000
```

ثم افتح المتصفح على:

```text
http://localhost:8000
```

## تعديل المحتوى
- غيّر النصوص داخل [index.html](index.html)
- غيّر الألوان والتصميم داخل [styles.css](styles.css)
- أضف أو عدّل سلوك الأزرار داخل [script.js](script.js)
